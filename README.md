# WEB KELAS SEDERHANA

Web yang dirancang untuk kenang kenangan kelas antara murid dan guru

## Fitur yang dimiliki

- carousel
- Anonymous Messaging App
- Ratting Slider

## Tujuan Web ini

Yaitu untuk membagikan foto kenangan antara murid dan guru disekolah selama 2 semester. disini juga menjadi pertengahan perjalanan dari jenjang pendidikan terakhir yaitu SMK dan disekolah SMK Maitreyawira Kota Dumai.

selain itu, web ini juga membantu saya untuk memahami dasar dasar dari html, css dan sedikit javascript, juga memahami framework css yaitu bootstrap untuk membantu membuat UI yang lebih menarik, dan membantu membuat web dalam jangka waktu yang singkat
